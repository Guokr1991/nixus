nixus

nixus is a program to obtain strain map from RF data.

Download

The CWEB documentation is available to download. The program is in a very raw state and any suggestions are welcome.

The files used in the project can be downloaded using git version control system:

$ # Unix command line
$ git clone git://dcm.ffclrp.usp.br/nixus/

Instalation

There is a Makefile script in the root directory of the project. The code is generated running make nixus.c and the documentation (LaTeX) with make nixus.tex or the command make docdoc to generate the pdf file.
